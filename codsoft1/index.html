Projects

Home Automation System

Overview My IoT-based home automation system integrates sensors for enhanced safety and convenience. It includes gas detection and soil moisture monitoring for a smart, responsive home environment. Key Features Gas Detection using MQ-8 Sensor: Utilizes the MQ-8 sensor to detect hydrogen gas. Monitors air quality and triggers alerts when gas levels are unsafe, preventing potential hazards. Smart Garden with Soil Moisture Detection: Uses soil moisture sensors to monitor plant hydration. Automatically activates watering or sends notifications when moisture levels are low, ensuring healthy plant growth and efficient water usage. This project demonstrates the effective use of IoT to automate home safety and gardening tasks, making daily life more efficient and secure.

Home Security System Using Raspberry Pi

My home security system project leverages Raspberry Pi to enhance home safety with motion detection and voice control capabilities. Key Features Motion Detection: Uses PIR (Passive Infrared) sensors connected to Raspberry Pi to detect movement. Triggers alerts and activates security cameras when motion is detected, providing real-time monitoring and recording. Voice-Controlled Home: Integrates voice recognition software with Raspberry Pi for hands-free control of security features. Allows users to arm/disarm the security system and control other smart home devices using voice commands. This project showcases the versatility of Raspberry Pi in creating an advanced, user-friendly home security solution.
